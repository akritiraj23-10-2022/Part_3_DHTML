How to create style sheet to alter an HTML element ?

The Style assignment process can be accomplished with the <STYLE>...</STYLE> tags. The syntax for making the assignment is simple. Between <STYLE> and </STYLE>, HTML tags to which styles are to be assigned are to be listed and each tag with a pair of curly brackets in which specific style attributes are to be specified. This <STYLE> tag should be within <HEAD>... </HEAD>element.

What are the attributes that can be specified ?

The attributes that can be specified are, Font Attributes, Color and Background Attributes, Text Attributes, Border Attributes, Margin Attributes and List Attributes.
